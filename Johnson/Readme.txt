**Code Description**
Implements Johnsons Algorithm to find all pairs shortest paths and outputs a matrix of all pairs shortest paths to output.txt. If there is a negative cycle it outputs negative cycle.
**Assumptions**
Input file contains graph represented as adjacency matrix; values of 2 million represent infinity
**How To Run**
You need to have run.sh file and your input file and the code files in the same directory and you can use ./run.sh for some input.txt
