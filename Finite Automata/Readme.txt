**Code Description**
Outputs starting indices of matches of a string with a pattern and reverse of the pattern using a finite automata
**Assumptions**
I assume only normal 26 english Alphabets are used; in lower case and all characters are on a single line with no spaces between for the pattern line and string line
I print out the indices in ascending order with spaces in the middle and no space after final index in output.txt
**How To Run**
You need to have run.sh files in the same directory and you can use ./run.sh for some input.txt
